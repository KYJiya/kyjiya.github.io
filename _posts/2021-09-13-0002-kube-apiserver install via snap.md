# Snap을 통해 kube-apiserver를 설치해 보자.
<br>
```
$ sudo snap install kube-apiserver
kube-apiserver 1.21.4 from Canonical✓ installed

```

## 확인
```
$ kube-apiserver --version
Kubernetes v1.22.2

```
